Steps to follow:

1. Start the development server with the command: python manage.py runserver
2. Use a tool like Postman or cURL to make requests to the API endpoints:
3. To create a blog post, send a POST request to blog/create/ with data in JSON format.
4. To update a blog post, send a PUT request to blog/update/{id}/
5. To view all the blog posts, send a GET request to blog/all/
